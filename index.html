Assignment Showcase

Showcase of Assignments

Upload your solved assignments to your GitHub repo (or provide public links), then add entries below to list them here. This static site is perfect for a portfolio or class resource.

How to add assignments:
  1. Place the assignment PDF/image inside assets/ (or host it elsewhere and paste the public URL).
  2. Edit js/script.js and add a new object to the assignments array with the file URL, title, subject, and description.
  3. Commit & push to GitHub. The list updates automatically from the array in script.js.
